This project was the core of my bachelor's thesis. I explored the challenge of accurately diagnosing glioblastoma from fMRI data, a crucial step in early intervention. The hybrid model was designed to combine the spatial feature extraction capabilities of ConvNeXt with the long-range dependency capture of Vision Transformers. Key features of the implementation include:

  • Advanced data preprocessing pipeline to handle 4D fMRI data.
  • Custom PyTorch Lightning module for efficient model training and validation.
  • Comprehensive evaluation using metrics beyond accuracy, such as AUC-ROC and F1-score, to ensure model robustness.
